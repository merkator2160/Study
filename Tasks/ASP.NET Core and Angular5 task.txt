Тестовое задание
Технологии 
- ASP.NET Core 
- Angular 2 or Angular 5 

Ориентировочное время выполнения
- 2 часа 

Необходимо: 
1. Создать пустое приложение Angular 2 или 5 + ASP.NET Core 
2. В Angular Добавить 3 компонента: меню, таблица, инструкции 
3. Добавить роутинг (переключение видов справа при нажатии на меню) 
4. На серверной стороне необходимо добавить 2 метода API: для получение данных таблицы и для получения текста инструкции. Данные в формате JSON 
5. Добавить простую БД, сделанную посредством EntityFramework
6. Дизайн не требуется, достаточно базовой верстки 
7. Редактирование/добавление/валидация данных не требуется 

ВАЖНО: в архив необходимо в положить файл read.me с интсрукцией на английском языке по запуску проекта

Результат можно прислать либо архивом, либо выложить на GitHub и прислать ссылку.
